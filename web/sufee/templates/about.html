Sufee Admin for Flask

Python Flask

Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions. And before you ask: It's BSD licensed!

Flask is Fun

from flask import Flask
        app = Flask(__name__)
        
        @app.route("/")
        def hello():
            return "Hello World!"

And Easy to Setup

$ pip install Flask
            FLASK_APP=hello.py flask run
         * Running on http://localhost:5000/

To Setup this Example Locally

git clone https://github.com/MikeRyan56/Flask-Sufee-Admin.git
          $ virtualenv env
          $ env\scripts\activate or sources env\bin\activate
          $ cd web
          $ set FLASK_APP=runserver.py
          $ flask run
           * Running on http://localhost:5000/

Run on Docker

docker pull mikeryan56/sufee-admin-flask
